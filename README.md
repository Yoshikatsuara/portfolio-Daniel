# Daniel Ara · Customer Success Analyst (Portfólio)

Portfólio pessoal focado em dados e relacionamento com cliente, projetado como um instrumento digital de prova técnica. Apresenta cases reais e métricas de impacto no formato STAR, construído com foco em legibilidade, contraste e fluidez cinematográfica (Midnight Luxe Adapted).

## 🛠 Stack Tecnológica
- **Framework:** Next.js 16 (App Router)
- **Estilização:** Tailwind CSS v4
- **Animações:** GSAP 3 + ScrollTrigger
- **Linguagem:** TypeScript

## 🚀 Como Rodar Localmente

1. Instale as dependências:
```bash
npm install
```

2. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

3. Para criar a build otimizada de produção:
```bash
npm run build
```

## 📂 Estrutura do Projeto

- `src/app/`: Rotas principais (`/`, `/styleguide`), metadados, sitemap e tokens de tema globais.
- `src/components/sections/`: O esqueleto da aplicação dividido em módulos lógicos (Hero, Cases, Stack, Philosophy, About, Contact, Footer).
- `src/components/ui/`: Micro-componentes reaproveitáveis de alta interação visual (MagneticButton, Badge, Navbar).
- `src/components/animations/`: Componentes que envolvem lógicas ricas de animação (`SplitText.tsx`).
- `src/content/`: Fonte de dados reais. Constantes fortemente tipadas contendo a trajetória profissional (`profile.ts`, `stack.ts`, `cases.ts`).
